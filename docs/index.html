Где взять Client ID
  1. Откройте Spotify Developer Dashboard.
  2. Создайте app и выберите Web API.
  3. Добавьте Redirect URI, показанный выше.
  4. Скопируйте Client ID из настроек app и вставьте сюда.
  5. Client Secret не вставляйте и никому не отправляйте.
Подробная настройка Last.fm
  1. Откройте last.fm и войдите в аккаунт. Если аккаунта нет, создайте его.
  2. Откройте настройки подключённых приложений Last.fm: last.fm/settings/applications.
  3. В блоке «Предлагаемые приложения» нажмите «Подключить» именно у «Скробблинг Spotify».
  4. Не выбирайте «Воспроизведение в Spotify»: это нужно для прослушивания музыки на Last.fm, а не для scrobbling.
  5. Разрешите Last.fm доступ к Spotify и вернитесь на страницу настроек.
  6. Включите любой трек в Spotify и проверьте, что он появился как текущий трек в вашем Last.fm профиле.
  7. Откройте last.fm/api/account/create и создайте API account.
  8. В поле Application name укажите OBS Spotify Widget.
  9. В Homepage укажите URL этого сайта. Callback URL можно оставить пустым; если Last.fm требует значение, вставьте URL этого сайта.
  10. Скопируйте только API Key. Shared secret этому виджету не нужен.
  11. Вставьте Last.fm username и API Key в поля выше и нажмите «Сохранить Last.fm».

Прогресс в Last.fm режиме приблизительный: длительность берётся из Last.fm, а позиция считается с момента, когда виджет увидел текущий трек. Если Last.fm не знает длительность, используется 3:00.
